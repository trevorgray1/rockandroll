# Rock & Roll Genealogy Project

This project is an encyclopedia and genealogy explorer for rock & roll musicians, bands, and their influences. It uses AWS for hosting, serverless backend, and graph database for relationships.

## Structure
- `frontend/`: Static site (React/Next.js)
- `backend/`: API code (Node.js, Python, or Lambda)
- `infra/`: Terraform scripts and modules for AWS resources
- `data/`: Data ingestion scripts and sample datasets
- `docs/`: Documentation and architecture notes

## Getting Started
1. Clone the repo
2. See `docs/README.md` for setup instructions
